HTML, que significa HyperText Markup Language, es el lenguaje estándar utilizado para crear y estructurar páginas web. Es un conjunto de etiquetas y elementos que definen el contenido y la disposición de una página web. En resumen, HTML proporciona la estructura básica que los navegadores web utilizan para mostrar el contenido de una página a los usuario
